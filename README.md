# Shark Byte 🦈

## 游戏简介

在这个赛博朋克风格的深海冒险中，你将扮演一条高科技鲨鱼，执行破坏海底光缆网的任务。你需要在避开 AWS、Azure 和 GCP 的防御无人机的同时，咬断尽可能多的光缆。

## 游戏特色

- 🎮 **流畅的 3D 控制**：控制鲨鱼在深海中自由游动
- 🌊 **沉浸式环境**：低多边形风格的深海世界，配合霓虹色调
- ⚡ **动态战斗**：多种类型的敌人，各具特色的攻击模式
- 🎵 **音效系统**：背景音乐和音效，可调节音量
- 🚀 **冲刺机制**：加速前进，咬碎光缆

## 操作说明

- **鼠标移动/摇杆**：控制鲨鱼游动方向
- **左键/DASH**：冲刺前进 / 咬碎光缆
- **右键/BACK**：后视镜模式
- **P 键/PAUSE**：暂停游戏
